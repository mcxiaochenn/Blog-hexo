## 初始化 Hexo 项目

完成以上环境后，就可以初始化本地 Hexo 项目了！

### 安装Hexo CLI

```shell
npm install -g hexo-cli
```

执行完成后，输入：
```shell
hexo -v
```
确认是否成功安装。

### 创建博客文件夹
```shell
mkdir my-blog
cd my-blog
```

### 初始化Hexo
```shell
hexo init
npm install
```

### 编译启动Hexo
```shell
hexo cl
hexo generate
hexo s
```

![shell显示如上即成功](/img/从零开始使用Hexo框架搭建属于你的博客/1/jietu/localhost4000.png)
